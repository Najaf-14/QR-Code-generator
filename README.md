# QR Code Generator

A simple web application that generates QR codes from text or URLs.

## Features

- Enter any text or URL
- Click "Generate QR Code" to create a QR code
- The QR code image appears below the input

## How to Use

1. Open `index.html` in your web browser
2. Type your text or URL in the input field
3. Click the "Generate QR Code" button
4. Your QR code will appear below

## Files

- `index.html` - The main HTML page
- `style.css` - Styling for the page
- `script.js` - JavaScript code for generating QR codes

## Requirements

- Internet connection (uses qrserver.com API)
- Modern web browser

## API

This app uses the free QR Server API: https://api.qrserver.com/
